# FUTURE INTERNS – PROMPT ENGINEERING TASK 2

## Task Title  
Build an AI-Powered Website Using a No-Code Tool

## Internship Domain  
Prompt Engineering (PE)

## Website Name  
MindFuel AI

## Website Description  
MindFuel AI is an AI-powered motivation and productivity website designed to help students and professionals stay disciplined, focused, and consistent. The website delivers motivational content, productivity tips, and self-growth guidance generated using AI.

## Tool Used  
- Lovable AI (No-Code AI Website Builder)  
- ChatGPT (Prompt creation and refinement)  
- GitHub (Project documentation)

## How AI Was Used  
- Prompts were used to generate the complete website structure and layout  
- AI generated motivational and productivity-focused content  
- AI-assisted design decisions ensured a clean and user-friendly experience  

## Key Features  
- AI-generated homepage with motivational messaging  
- Daily motivation and productivity tips  
- About page explaining AI-driven content creation  
- Contact form for user interaction  

## Live Website  
(https://mindfuel-ai-hub.lovable.app/)

## Learning Outcome  
This task helped me understand how prompt engineering can be applied to real-world applications such as no-code website development, enabling rapid creation of AI-powered digital products.
